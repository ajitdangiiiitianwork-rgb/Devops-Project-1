# AWS Resource Tracker – DevOps Project 1

## 📌 Overview
This project contains a shell script (`aws_resource_tracker.sh`) that helps track AWS resources in a concise format.  
It lists EC2 instances, S3 buckets, Lambda functions, and IAM users, making it easier to monitor cloud usage.

## 🚀 Features
- List all S3 buckets
- List EC2 instances (IDs only for clarity)
- List Lambda functions
- List IAM users
- Clean, readable output format

## 🛠 Prerequisites
- AWS CLI installed
- AWS credentials configured (`aws configure`)
- Access to an EC2 instance or local environment with AWS CLI

Run the script:
```bash
./aws_resource_tracker.sh
